# 一、学习目标

本周实验旨在通过一系列循序渐进的 Python 编程任务，使学生掌握科研编程的基本规范与思维方式，为后续生物建模与数据分析打下坚实基础。

**主要学习目标包括：**
- **理解并应用 Python 程序的标准结构**
- **掌握脚本的规范写法**（shebang、文档字符串 docstring、main() 函数结构及命令行参数解析，形成良好的科研代码风格）
- **熟悉循环、条件控制与列表推导式等核心语法**
- **掌握常用数据结构与文件操作方法**（list、tuple、dict、文件读写 open/Path）
- **将编程应用于典型生物学问题建模**（DNA 序列比对与 Lotka–Volterra 捕食者—被捕食者模型）
- **学习科学计算与性能优化思想**（NumPy 向量化、cProfile 性能分析）
- **培养数据可视化与结果呈现能力**（matplotlib 绘图、PDF 输出）
- **了解科研编程的跨语言比较与协同工作流程**（Python 与 R 性能对比）
- **掌握 Git 版本控制与协作式科研开发流程**（test → add → commit → push 四步法）

# 二、主要任务与实现

## 1. 基础结构与模板设计：`boilerplate.py`
- 建立统一代码模板，展示科研编程标准结构：
  - shebang 与 docstring（脚本用途、作者、版本）
  - main() 函数与 `if __name__ == '__main__'` 入口
  - 命令行参数读取与文件输出（sys.argv、Path）
  - 模块化设计（函数与主程序分离，便于复用）

## 2. 控制流与数据结构练习：`cfexercises1.py`
- 改写为模块化程序，所有 foo_x 函数接收参数
- 涵盖条件判断、循环、递归、阶乘计算
- 脚本末尾加入测试调用，所有输出经 main() 控制

## 3. 循环与推导式练习：`lc1.py`、`lc2.py`、`dictionary.py`、`tuple.py`
- 强化数据结构与推导式掌握
- 列表/字典推导式、字典合并、键值计数与排序、元组打包解包

## 4. 序列比对项目：`align_seqs.py`
- 从 CSV 文件读取 DNA 序列，滑动比对，输出最佳得分与对齐结果

## 5. 连续时间捕食者—被捕食者模型：`LV1.py`
- Lotka–Volterra 微分方程，RK4 积分，PDF 绘图

## 6. 密度制约模型：`LV2.py`
- 引入资源密度依赖项 K，命令行参数化，输出稳定种群与参数标注图

## 7. 脚本性能剖析：`run_LV.py`
- 自动化测试与性能分析（subprocess、time、cProfile）

## 8. 离散时间与随机性扩展：`LV3.py`、`LV4.py`
- 差分方程、环境噪声模拟，PDF 输出，性能剖析

## 9. 向量化效率比较：`Vectorize1.py`、`Vectorize2.py`、`compare_vectorization.sh`
- NumPy 向量化与循环性能对比，跨语言实验

## 10. 正则表达式练习：`RegExercises`
- re 模块姓名、日期、模式匹配，鲁棒性与扩展性探讨

## 11. 版本控制与协作提交
- 测试、添加、提交、推送，保证可追溯性与复现性

# 三、结果与分析

## 1. 脚本功能与运行结果
- 所有脚本可在无图形界面环境下运行，输出文件至 results/ 目录
- 统一模板结构，命令行参数化，良好封装与可移植性

## 2. 基础语法与数据结构练习
- 多种实现方式成功计算阶乘，所有函数正确处理输入
- 列表推导式效率高于传统循环，代码简洁
- 字典与元组操作均返回预期结构

## 3. DNA 序列比对结果
- 成功读取两条 DNA 序列并比对，输出最佳匹配结果
- 算法自动化、可扩展，能处理上千碱基

## 4. 连续时间 Lotka–Volterra 模型
- 周期振荡行为与理论一致，PDF 图表清晰展示系统动态

## 5. 密度制约 LV 模型
- 携带容量 K 决定系统稳定性，参数变化影响种群共存或灭绝

## 6. 离散时间 LV 模型
- 合理参数下振荡趋于平稳，高增长率易失稳，呈现离散混沌特征

## 7. 离散随机 LV 模型
- 噪声影响系统稳定性，模拟环境波动扰动作用

## 8. 性能剖析结果
- 连续模型略快于离散模型，主要瓶颈为积分循环与绘图
- NumPy 向量化可进一步优化

## 9. 向量化性能比较
- 向量化操作平均快 50–70 倍，NumPy 底层优势明显

## 10. R 与 Python 跨语言比较
- 循环性能差距大，向量化后接近持平，Python 更灵活

## 11. 正则表达式练习结果
- 改进表达式能识别复杂姓名与日期，掌握文本规范化技能

## 12. 版本控制与输出验证
- 所有结果文件成功生成，Git 历史完整，命名规范
